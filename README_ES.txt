Instrucciones para Windows / Windows Intructions (Spanish)

1. Descargar e Instalar Git: https://git-scm.com/downloads
2. Descargar Python 2.7: https://www.python.org/downloads/
3. Instalar en C:\Python27
4. Click derecho en Equipo 
   > Propiedades 
   > Propiedades Avanzadas
   > Opciones Avanzadas
   > Variables de Entorno
   > Buscar una variable del sistema llamada "Path"
   > Agregar a esta la ruta:
     "C:\Python27; C:\Python27\Scripts"
   > Aceptar
5. Abrir la consola de Git:
6. Ejecutar: 
   "git config --system http.sslverify false"
7. Descargar el proyecto con:
   "git clone https://github.com/64lines/project_eba"
8. Instalar Django:
   "pip install django"
9. Ejecutar el servidor local:
   "python manage.py runserver"
10. Verificar el servidor local ingresando en el navegador
   "127.0.0.1:8000/admin"
11. Intalar Sublime Text: http://www.sublimetext.com/2
12. Abrir Sublime Text y cargar la carpeta del proyecto en "/Users/nombredelusuario" o "/Usuarios/nombredelusuario".
10. Disfrutar!
